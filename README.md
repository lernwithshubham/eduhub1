# Eduhub1

## Purpose
Eduhub1 is an educational Java project focused on media handling, providing support for both audio and video modules. This project aims to facilitate learning through interactive media engagement and help students understand the basics of media processing in Java.

## Repository Structure
The repository consists of the following structure:

- **src/**: Contains all the source code files for the project.
  - **audio/**: Module dedicated to audio processing functionality.
  - **video/**: Module dedicated to video processing functionality.
- **lib/**: Libraries and dependencies used in the project.
- **docs/**: Documentation files and resources to assist users in understanding the project.
- **tests/**: Unit tests for verifying the functionality of the modules.

## Getting Started
To get started with the project, clone the repository and follow the instructions in the documentation located inside the `docs/` directory for setting up the development environment and running the application.
